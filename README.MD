# Como executar o projeto


## Front end web
Pré-requisitos: npm / yarn

```bash
# clonar repositório
git clone https://github.com/MatheuSilvaSant/netflix

# entrar na pasta do projeto front end web
cd front-web

# instalar dependências
npm install

# executar o projeto
npm start
```
